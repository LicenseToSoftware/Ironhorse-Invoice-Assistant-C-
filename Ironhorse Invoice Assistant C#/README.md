# Ironhorse Invoice Assistant

## Project Structure

- `Forms/` – Windows Forms (UI screens)
- `Services/` – Core logic such as image resizing or invoice generation
- `Helpers/` – Utility functions (e.g., file access, formatting)
- `Models/` – Data structures representing invoices, images, etc.
- `Resources/` – Static files like icons or embedded images
- `Output/` – Folder where resized or compressed files are saved
